# Week 02: Physiological Signal Processing

## Focus

MATLAB analysis of ECG/EMG/PPG-style physiological signals, including sampling, filtering, and basic signal interpretation.

## Public Contents

- `code/` contains the MATLAB analysis script.

## Excluded Contents

Raw personal physiology recordings and derived personal waveform plots are not included. Submitted report files and course handouts are also excluded.

## Reproducibility Note

The script may require local signal files from the original lab session. Those files are intentionally omitted from the public repository.
