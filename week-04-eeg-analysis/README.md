# Week 04: EEG Analysis

## Focus

EEG processing for task-based analysis, filtering, and band-power interpretation.

## Public Contents

- `code/` contains MATLAB scripts for blink, eye-open/closed, facial-expression, head-movement, math-task, filter-design, and segment-filtering analyses.

## Excluded Contents

Raw EEG recordings, personal task data, derived personal plots, submitted reports, and copied course materials are excluded.

## Reproducibility Note

The scripts are kept to show the processing methods. Full reruns require private EEG files from the lab session.
