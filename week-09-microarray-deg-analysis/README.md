# Week 09: Microarray DEG Analysis

## Focus

Microarray preprocessing and differential expression analysis using MATLAB.

## Public Contents

- `code/` contains the MATLAB DEG analysis script.
- `results/` contains selected figures and output tables for normalization, expression comparison, t-test results, and top DEG lists.

## Excluded Contents

Duplicated raw expression matrices, submitted reports, and submission archives are excluded.

## Reproducibility Note

The tracked result files document the analysis output. Full reruns may require original course datasets that are not included.
