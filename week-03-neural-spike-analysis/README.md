# Week 03: Neural Spike Analysis

## Focus

Neural spike detection and stimulation-response analysis using MATLAB filtering, thresholding, and multi-channel signal processing.

## Public Contents

- `code/` contains the MATLAB scripts for multi-channel spike detection and stimulus-response analysis.

## Excluded Contents

Large raw neural recording files and submitted reports are excluded from the public repository.

## Reproducibility Note

The code documents the analysis workflow, but rerunning it requires the original neural recording files, which are not published.
