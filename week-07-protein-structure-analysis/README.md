# Week 07: Protein Structure Analysis

## Focus

Protein sequence alignment, FASTA database search, and comparison of structure-modeling outputs.

## Public Contents

- `code/` contains Python alignment code and a sample notebook.
- `data/` contains small FASTA inputs used for the alignment examples.
- `results/` contains selected non-identifying figures and PDB model files from BLAST, SWISS-MODEL, AlphaFold, and structure-review steps.

## Excluded Contents

Submitted reports, copied course instructions, and duplicated submission archives are excluded.

## Reproducibility Note

The included data are small enough for public review. External web-tool outputs may differ if rerun later because databases and prediction services can change.
