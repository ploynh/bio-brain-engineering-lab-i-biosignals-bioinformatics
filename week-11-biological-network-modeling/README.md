# Week 11: Biological Network Modeling

## Focus

ODE-based biological network simulation, including Michaelis-Menten-style phosphorylation behavior and a self-contained MAPK cascade model.

## Public Contents

- `code/` contains MATLAB model and parameter-sweep scripts.
- `results/` contains selected non-identifying simulation figures.

## Excluded Contents

Submitted reports, archived submissions, and course support packages are excluded.

## Reproducibility Note

The tracked MATLAB files are self-contained examples of the retained modeling work. Scripts that depended on omitted course support files were not published.
